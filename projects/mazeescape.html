About

Maze Escape was a fun project inspired by moves such as The Maze Runner. The premise was a multiplayer, procedural maze that the players needed to escape from. The maze would be filled with enemies and traps as well as puzzles. I also wanted the ability for the maze to change at certain intervals. Because of these requirements, I developed a procedural system that generates a maze within a given boundary. The exit is traced first before applying recursion to place the remaining maze tiles. The script allows for any type of tiles to be placed, including ones with different shapes or textures applied to them. My goal was to have quite a bit of variety throughout the maze.

Current Development State

Unfortunately, the game itself wasn't very fun to play (at least in single player). In the movie, the maze was designed to be torture, and I feel that this project accomplishes the same thing. Running around trying to escape with 1 life is not very enjoyable. With a large maze, it could potentially take over an hour for the user to escape in a single-player game. There are quite a few things that I'd add in order to make the game more enjoyable if I ever resumed this project.

Update: Apparently I have lost the latest build (with working AI). The uploaded build is an earlier proof of concept that includes maze generation and the ability to escape/start new games.
