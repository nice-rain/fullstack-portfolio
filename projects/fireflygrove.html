Background

Firefly Grove is a Farming/Exploration game that was inspired by games such as Harvest Moon. Instead of eveloping a top-down 2D tile-based game, I opted for a more realistic 3D approach. As a player, you have acquired a home and a small plot of land on a planet that has recently been terraformed. Many of them reject technology and are weary of strangers. You will need to farm, mine, explore, craft, and sell in order to build a new life on this planet. While the game is currently on home, I do intend to eventually pick up development and finish the project.

Features

  • Farming system complete with 4-5 different plants
  • Fully working inventory/storage as well as a sales system for income
  • Dialogue system with multiple dialogue paths
  • Functioning quest system that allows for repeat quests and chained quests
  • Day/Night Cycle
  • Farm Upgrade system for placing things such as a well or barn
  • Simple fishing system
