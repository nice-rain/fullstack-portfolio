About

Unreal Hue is a project that interfaces Unreal Engine with a Philips Hue system. This allows for controlling individual light intensity, as well as light colors and affects. Because I only had white bulbs to test with, color was not implemented in this project. The project is pretty dirty and everything was hard-coded in order to work with the API on my home device. In the future, I plan to move the logic into an engine module and release it.
