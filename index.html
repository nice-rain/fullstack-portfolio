About

I have always been excited to learn and create new things. My programming background started off at a young age learning how to modify online games. I created several mods for games like Starsige: Tribes and worked on several small projects with friends using Torque Game Engine Advanced. I eventually moved to Dallas and worked as a programmer for an indie game development company for 2 years.

My interest in full stack development started when I began noticing that mobile applications, websites, and games were all able to display the same content delivered by a server. I wanted to learn how to setup and create back-end and front-end systems. I joined Thinkful in July, and in the time that I have been studying, I have learned HTML, JavaScript, CSS, and JQuery. Currently, I am studying Node.js and MongoDB. I love building new things and enjoy working with others.

Now that you know a little more about me, contact me, and let's create something amazing!
